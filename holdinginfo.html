Phuket Marriot Resort

Project Case Study

Scope :

Redesign & develop a prototype website for the Phuket Marriott Hotel & Resort, Thailand. Providing an updated code base, responsive design patterns, accessibility options & much more.

Developed to showcase what this amazing resort has to offer in terms of facilities, services, restaurants & beach access. The current website doesnt do the resort justice with a bland interface jumbled together.

Objectives :
  • Clean & reusable code structuring
  • Component level CSS styling
  • Vanilla Javascript interactivity
  • Responsive Design features
  • Updated color scheme & images
Build Tools
  • HTML
  • CSS
  • JS
  • PHP
  • Netlify
