Phuket Marriott Resort

After staying at the Phuket Marriott Resort & Spa last year and seeing how the webiste did not reflect the experience of the resort. I decided to create & develop a redesign for the website to reflect and share this amazing the experience.

The redesign enhances the previous design by adding responsive elements to the layout, providing a welcoming and information landing page, and showcasing the resorts world class facilities, services & suites.

Travel Highlights App

When travelling its always a great idea to be able to save desintations, locations, places of interest & great restaurants. I developed a web application that enables users to create a travel highlights visual database.

The application can add a new country or city panel, with provides a heading and a tagline for the user to name. Add highlight images, favourite places & best deals for food. Best of all, you can leave a small quote about what you enjoyed about a particualr area for other travellers to view.

Dashboard JS

Developed a dashboard to help keep track of personal projects, research & documentation for learning materials, and providing an indepth overview and insight to planning for overseas travel.

The dashboard web application has a robust set of features, enabling users to have complete control over particular sections of the dashboard, hiding, removing & adding features at the users request. And keeping track of unfinished tasks or debugging code bases.

Display Folio

A minimalistic online digital folio template for university students to help display work, projects & skills to potential employers. Started out as a personal portfolio for a friend, has now been turned into a templating system, with a multitude of features available.

Add a display image or display content in the main container. Choose a title, description, create links and social connections, as well as previewing and screen capturing technology.
